# Sulaiman's Professional Portfolio 
--
## Tech Stack
*   **Backend:** PHP
*   **Database:** MySQL
*   **Frontend:** HTML5, CSS3, Vanilla JavaScript
*   **Tools:** SonarQube (Static Analysis), Git/GitHub
--
## Featured Projects
1.  **Student Marks Management System:** A full-stack application for managing student academic records.
2.  **Task Timer:** A productivity tool built with Vanilla JS to track task durations.
3.  **Responsive CV:** A professional personal profile built with clean HTML/CSS.
--
## How to Run Locally
1. Clone the repository.
2. Import the `database/portfolio_db.sql` file into your MySQL server.
3. Update `db_config.php` with your local credentials.
4. Run via XAMPP/WAMP.
